# 刘佳飞 博士

副教授 / 教研室主任 / 硕士生导师（学硕、专硕）

## 个人简介

刘佳飞（1993-），河南洛阳人，中共党员，工学博士，硕士研究生导师，入选广西八桂青年拔尖人才、广西第一批青苗人才计划、广西青托人才工程（资助类），荣获广西师范大学优秀班主任、本科教学优秀奖、教学新秀等。

主要从事**容错计算、故障诊断、网络优化以及基于社交网络的大数据分析**等方面的研究。主持国家自然科学基金青年项目（29岁获得）、广西自然科学基金及其他各类科研项目9项，参与国家级项目4项。在IEEE TC、IEEE/ACM ToN、IEEE TR、IEEE TCBB、ESWA、JPDC、INS、COCOON等计算机领域国际权威期刊或会议中发表学术论文40篇以上。

担任IEEE TC、IEEE ToN、IEEE TR、JPDC、TCS、DAM、计算机科学等期刊审稿人、广西师范大学学报（自然科学版）青年编委。CCF理论计算机科学执行专委、国家自然科学基金项目评审专家、中国计算机学会(CCF)会员、中国人工智能学会(CAAI)终身会员、中国工业与应用数学学会(CSIAM)会员、CCF YOCSEF 桂林分论坛学术委员、副主席，CCF桂林分部执行委员。
